# Bills Bills Bills — Wine Club Churn Analysis

Project scaffold for churn analysis and retention modeling.

Structure:
- `data/` — raw and derived datasets (raw, dim, processed, analytical)
- `notebooks/` — exploratory and modeling notebooks
- `sql/` — SQL scripts for dimensions, facts, and analytical queries
- `outputs/` — charts, tables, and member lists
